HTML学习笔记

发表于 2023年5月10日

今天学习了HTML的基础知识，以下是我的学习笔记：

HTML基本结构

每个HTML文档都应该包含以下基本结构：

<!DOCTYPE html>
<html>
<head>
    <title>页面标题</title>
</head>
<body>
    页面内容
</body>
</html>

常用HTML标签

  • <h1> 到 <h6>: 标题
  • <p>: 段落
  • <a>: 链接
  • <img>: 图片
  • <ul> 和 <li>: 无序列表
  • <ol> 和 <li>: 有序列表

HTML属性

HTML元素可以有属性，属性提供了关于元素的额外信息。例如：

<a href="https://example.com">这是一个链接</a>
<img src="image.jpg" alt="描述文字">

学习HTML是创建网页的第一步，接下来我将继续学习CSS来美化我的网页。
